Buttersandwich UI

A study of modern CSS, new CSS features (be them widely supported or not), best practices and advanced grid layouts.

This project serves as a showcase and playground for the most modern HTML/CSS techniques available, without any Javascript. Most of the modern CSS used here will not be supported by most browsers. Even Chrome can't handle some of them natively - and to do so, we must:

  1. Go to chrome://flags
  2. Search for "Experimental Web Platform features".
  3. Set it to Enabled and relaunch.
